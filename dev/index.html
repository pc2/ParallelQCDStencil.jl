Installation

] add https://git.uni-paderborn.de/cs-hit/theses/2022_parab/qcd-stencil/ParallelQCDStencil.jl

Note: The minimal required Julia version is 1.7.

Usage

julia> using ParallelQCDStencil

Example

Lattice with 3*3 Index matrix link values

d = 2       #diamensions
L = 5       #number of lattices in per diamension
N = L^d     #number of sites
NL=N*d      #number of links

#input
link_linear_data=[I(3) for i in 1:NL]

#formatting input into array fashion
fi=input_formatting(d,L,N,link_linear_data)

compute_stencil(N,d,L,fi)
