ParallelQCDStencil.jl

Installation

] add https://git.uni-paderborn.de/cs-hit/theses/2022_parab/qcd-stencil/ParallelQCDStencil.jl

Note: The minimal required Julia version is 1.7.

Assumptions

  • Link matrices are of same type and of size 3x3 (independent of dimension)
  • For parallel
    • Local grids must be square and of same size
    • (global grid/ local grid) must be an integer

Usage

julia> using ParallelQCDStencil

Input parameters

We are using struct for passing around the parameters to the fuctions. It consists of :

L = 5                           #number of lattices in per dimension
N = L^d                         #number of sites
NL = N * d                      #number of links
LD = 3                          #link matrix length
C1::T = 2                 #stencil constant 
C2::T = 3                 #stencil constant   
IG = 2                          #local grid length
n_IG_d = Int(floor(L / IG))     #grids per dimension
n_IG = n_IG_d^d                 #number of local grids
master_rank = 0                 #mpi master rank

The struct can be construced using :

p = StencilPara{Int64, 2}(L=4, IG=2, master_rank=0)

Input data

For stencil computation input can be:

  • vector of link data ()
rmat = rand(ComplexF64,(3,3))  
data = [rmat for i in 1:NL]
  • HDF5 file path :
    • File containing array with variable name "data"
    • e.g. 2D array of size 10x10x2x3x3

Basic Example with array input

#basic_example.jl
using MPI
using Parameters
using LinearAlgebra
using Test

p = StencilPara{Int64, 2}(L=10, IG=5, master_rank=1)
@unpack NL, master_rank, C1, C2, N = p
MPI.Init()
comm = MPI.COMM_WORLD
rank = MPI.Comm_rank(comm)

if (rank == master_rank)
    rmat = [
        0.630657+0.343875im 0.682196+0.797296im 0.641287+0.931409im
        0.566447+0.395339im 0.186153+0.685137im 0.484366+0.325613im
        0.561107+0.889588im 0.978503+0.803343im 0.69018+0.652149im
    ]
    data = ParallelQCDStencil.linear_link_values_to_site_based_array(p, [rmat for i in 1:NL])
else
    data = nothing
end
total = compute_stencil(p, data, comm, rank, LocalGridNaive())

if (rank == master_rank)
    println("total:$(total)\n")
end
MPI.Finalize()

Basic Example with file input

using ParallelQCDStencil
using MPI
using Parameters
using LinearAlgebra
using Test
using HDF5

p = StencilPara{Int64, 2}(L=10, IG=5, master_rank=1)
@unpack NL, master_rank, C1, C2, N, L, d = p

MPI.Init()
comm = MPI.COMM_WORLD
rank = MPI.Comm_rank(comm)
file = "2d_array_l_10_cf64.h5"
total = compute_stencil(p, file, comm, rank, LocalGridNaive(); file=true)

if (rank == master_rank)
    println("total:$(total)\n")
end
MPI.Finalize()

Code execution

The program can be launched using MPI command e.g:

mpirun -n 4 julia --project basic_example.jl
